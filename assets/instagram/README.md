# Assets Instagram @nakesproid

```
assets/instagram/
├── templates/         ← Master template Canva (png preview)
├── screenshots/       ← Screenshot website client
├── batch-1/           ← 4 postingan perdana
├── batch-2/           ← Postingan minggu 2
├── story/             ← Desain story harian
└── hasil-jadi/        ← Arsip setelah posting
```

## Cara Pakai
1. Screenshot website client → simpan di `screenshots/`
2. Desain di Canva pake master template
3. Export .png → taruh di `batch-1/` atau `batch-2/`
4. Posting ke IG
5. Arsip ke `hasil-jadi/`
